The Difference Between Web1,Web2 And Web3:

  • In web1 the data the user interact with is mostly static and in web2 the focus on the data is that it will be more dynamic and in web3 the focus on the data is the difference representation of the same data on different devices.

  • In web1 the data representation is mostly in a form of a web form and in web2 the data representation is mostly in a form of a web application and in web3 the data representation is mostly in a form of a smart application.

  • In web1 the data retrieved from the server file system while in web2 the flows between site owner and site users and in web3 the focus is on the semantics of the data.
